3.main：網頁的主要內容。

[比較] .gitignore 會真的忽略的檔案

  • 已經追蹤的檔案(已 commit) 改變之後還是追蹤

    • 修改內容之後 依然會有紀錄

  • 建立之後馬上加入.gitignore (至少是在 add 前

    • 就可以正確避免被追蹤

  • 想消除追蹤 詳情:
    Ignore files that have already been committed to a Git repository
